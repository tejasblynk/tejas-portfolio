MULTILEVEL SMART PARKING GRID

ESP32 IR Arrays Servo Mechanics

An urban infrastructure prototype addressing parking congestion. It manages a 6-slot bay with automated entry denial if capacity is reached.

TECHNICAL SPECIFICATIONS:

  • Controller: ESP32 WROOM (High processing power)
  • Sensor Array: 2x IR (Gate) + 6x IR (Slot Occupancy)
  • Actuator: SG90 Servo Motor (Boom Barrier)
  • Feedback: LCD I2C 20x4 + Active Buzzer (Entry/Exit tones)
  • Algorithm: Counter variable increments on entry, decrements on exit. Locks gate at count=6.
