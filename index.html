IoT Smart Agriculture System

Automatic Plant Watering Gadget

NodeMCU ESP8266 Capacitive Soil Sensor v2.0 2-Channel Relay LCD I2C 1602 SinricPro IoT Alexa Enabled

Designed to assist farmers and busy individuals, this system intelligently manages plant hydration. It utilizes a capacitive soil moisture sensor to read ADC values, determining soil dryness.

  • Auto Mode: Automatically triggers the submersible pump via relay when soil is dry based on sensor thresholds.
  • Manual/Voice Mode: Full pump control via the SinricPro mobile app or voice commands through Alexa integration.

Multilevel Smart Parking System

Urban Infrastructure Solution

ESP32 Controller 8x IR Sensors SG90 Servo Barrier LCD I2C 20x4 Active Buzzer

Address urban parking scarcity and fuel wastage. This 6-slot system provides real-time visual feedback of empty slots on a large 20x4 LCD display.

  • Automated Entry/Exit: Two IR sensors control the SG90 servo boom barrier. The barrier locks if all slots are full.
  • Slot Detection: Six individual IR sensors monitor occupancy for each parking bay.
  • Audio Feedback: Buzzer alerts for entry, exit, and "Parking Full" status.
